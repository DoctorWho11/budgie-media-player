Music-player

A GNOME3.10 music player written in C

Download this project as a .zip file Download this project as a tar.gz file

Simple Music Player using GTK3.10. Tries to be pretty. Probably not very useful for some time yet. Written in C. Now supports video.

Why C?

That question has never needed answering.

Naming

Couldn't think of an awesome name, so for now we call this 'MusicPlayer'

Inventive++

Features

  • Lightweight GTK3.10 media application
  • Very fast database for fast startup time
  • Uses GStreamer-1.0
  • Supports audio & video
  • Modern UI design
  • Client side decorations
  • Random/shuffle function, that is actually random. Not the track you just played.
