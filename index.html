budgie

A GNOME3.10 media player written in C

It Plays Music. Oh, and videos.

See? Plays videos. Or stalks rabbits. Easter egg..?

Installation

The official build of Budgie is hosted on the openSUSE build service. Budgie is designed and built using openSUSE 13.1 so this is our official repository.

openSUSE https://build.opensuse.org/package/show/home:ikeydoherty/budgie

Arch Linux (Many thanks to Daniel!) https://aur.archlinux.org/packages/budgie-git/

Features (v1)

  • Lightweight GTK3.10 media application
  • Very fast database for fast startup time
  • Uses GStreamer-1.0
  • Supports audio & video
  • Modern UI design
  • Client side decorations
  • Random/shuffle function, that is actually random. Not the track you just played.

v1-v2

  • Finalize error handling
  • Allow playback control in full screen mode ✓
  • Allow manual re-import of library (instead of deleting database) ✓
  • Allow configuration of directories and setting persistence (partial support)
  • Support track-seeking (skip/seek) ✓

v2-v3

  • Album art
  • Stable database format (final)
  • Full Wayland support

v3-v4

  • iPod support
  • Rip media from various sources
